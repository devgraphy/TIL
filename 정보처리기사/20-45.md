14. JAVA: 10을 2진수로 변환

```java
public class Gisafirst { 
  public static void main(String[] args) {  
   int a[] = new int[8];
   int i = 0, n = 10;
   while (**n>0**) { 
     a[i++] = **n%2**; 
     n /= 2; 
   } 
   for (i=7; i>=0; i--)
     System.out.printf("%d", a[i]);
   }
}
```
